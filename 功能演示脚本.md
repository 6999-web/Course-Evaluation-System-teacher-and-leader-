## 常见问题

### Q: 如何重置密码？
A: 目前系统不支持自助重置密码，请联系管理员。

### Q: 如何导出评价报告？
A: 在评价报告页面，选择报告类型后，点击"导出"按钮选择格式。

### Q: 如何提交申诉？
A: 在"我要申诉"页面，点击"提交申诉"按钮，填写申诉原因并上传佐证材料。

### Q: 自动评分需要多长时间？
A: 单个文件评分通常在10秒内完成，批量评分（100份）约需10分钟。

## 系统架构

### 前端技术栈
- Vue 3.5.13
- TypeScript
- Element Plus
- Vite
- Socket.io

### 后端技术栈
- FastAPI 0.121.2
- SQLAlchemy 2.0.45
- Python 3.13
- Uvicorn
- Deepseek API

### 数据库
- SQLite (开发环境)
- MySQL (生产环境)

## 性能指标

- 单文件评分响应时间: ≤10秒
- 批量评分（100份）: ≤10分钟
- 自动与人工复核一致性: ≥95%
- API平均响应时间: <500ms
