## 🎓 AI评分使用

### 单个评分
```
1. 登录 → 考评任务管理
2. 找到"已提交"任务
3. 点击"AI自动评分"(蓝色按钮)
4. 等待10-30秒
5. 查看评分结果
```

### 批量评分
```
1. 勾选多个"已提交"任务
2. 点击"AI批量自动评分"
3. 确认对话框
4. 等待完成
5. 查看统计结果
```
