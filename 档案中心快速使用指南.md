# 档案中心快速使用指南

## 🚀 快速开始

### 访问档案中心
1. 登录管理端: http://localhost:5173
2. 点击左侧菜单"档案中心"
3. 选择"评分结果归档"标签页

---

## 📋 功能说明

### 1. 查看归档列表

**显示内容**:
- 归档ID
- 教师ID和姓名
- 考评表名称
- 得分和得分率
- 学期
- 归档时间

**筛选功能**:
- 按学期筛选
- 按考评表名称筛选
- 按教师ID筛选

**操作**:
```
1. 输入筛选条件
2. 点击"查询"按钮
3. 点击"重置"清空筛选条件
```

---

### 2. 查看详情 👁️

**操作步骤**:
```
1. 找到要查看的记录
2. 点击右侧"查看详情"按钮
3. 在弹出的对话框中查看完整信息
```

**详情内容**:

#### 📋 基本信息
- 归档ID、任务ID
- 教师ID、教师姓名
- 考评表名称
- 学期
- 评分时间、归档时间

#### 🏆 得分情况
- 总分（大数字显示）
- 得分率（百分比）
- 等级标签（优秀/良好/中等/合格/不合格）
- 得分进度条（彩色）

#### 📊 各项得分明细
- 评分项名称
- 得分 / 满分
- 得分率进度条

#### 💬 评分反馈
- 总体评价
- 专业反思深度
- 改进措施可操作性
- 专业发展规划

**对话框操作**:
- 关闭: 关闭对话框
- 导出详情: 导出当前评分的详细信息
- 打印: 打开打印预览

---

### 3. 导出功能 📤

#### 单个导出

**操作步骤**:
```
1. 找到要导出的记录
2. 点击右侧"导出"按钮
3. 文件自动下载到本地
```

**文件信息**:
- 格式: CSV
- 编码: UTF-8
- 命名: `评分详情_教师姓名_考评表名称_时间戳.csv`

**导出内容**:
- 基本信息
- 得分情况
- 各项得分明细
- 评分反馈

#### 批量导出

**操作步骤**:
```
1. 勾选要导出的多条记录
2. 点击表格上方"批量导出"按钮
3. 文件自动下载到本地
```

**文件信息**:
- 格式: CSV
- 命名: `评分归档_批量导出_时间戳.csv`

**导出内容**:
- 所有选中记录的汇总信息
- 每条记录一行

**提示**:
- 选中数量会实时显示在按钮上
- 支持跨页选择
- 可以用Excel打开CSV文件

---

### 4. 删除功能 🗑️

#### 单个删除

**操作步骤**:
```
1. 找到要删除的记录
2. 点击右侧"删除"按钮
3. 在确认对话框中点击"确定"
4. 等待删除完成
```

**注意事项**:
- ⚠️ 删除操作不可恢复
- ⚠️ 请谨慎操作
- ✅ 删除前会弹出确认对话框
- ✅ 删除成功后列表自动刷新

#### 批量删除

**操作步骤**:
```
1. 勾选要删除的多条记录
2. 点击表格上方"批量删除"按钮
3. 在确认对话框中点击"确定"
4. 等待删除完成
```

**提示**:
- 选中数量会实时显示在按钮上
- 显示成功和失败数量
- 失败的记录会在控制台显示错误信息

---

### 5. 打印功能 🖨️

**操作步骤**:
```
1. 打开评分详情对话框
2. 点击底部"打印"按钮
3. 在新窗口中预览打印内容
4. 点击"打印"按钮
5. 选择打印机和设置
6. 完成打印
```

**打印内容**:
- 标题: 教师评分详情报告
- 基本信息表格
- 得分情况（大字号）
- 各项得分明细表格
- 评分反馈
- 页脚: 打印时间、系统名称

**打印技巧**:
- 可以选择"打印为PDF"保存
- 可以调整打印设置（页边距、缩放等）
- 适合A4纸打印

## ❓ 常见问题

### Q1: CSV文件乱码怎么办？
**A**: 
1. 使用Excel的"数据" → "从文本/CSV"导入
2. 选择UTF-8编码
3. 或者使用WPS Office打开（自动识别编码）

### Q2: 批量导出有数量限制吗？
**A**: 
- 没有硬性限制
- 建议单次不超过1000条
- 大量数据建议分批导出

### Q3: 删除的记录可以恢复吗？
**A**: 
- 不可以，删除是永久性的
- 建议删除前先导出备份
- 重要记录建议保留

### Q4: 打印时内容显示不全怎么办？
**A**: 
1. 调整浏览器的打印缩放比例
2. 选择横向打印
3. 调整页边距
4. 使用"打印为PDF"功能

### Q5: 可以导出为Excel格式吗？
**A**: 
- 当前版本支持CSV格式
- CSV文件可以直接用Excel打开
- 在Excel中可以另存为.xlsx格式
