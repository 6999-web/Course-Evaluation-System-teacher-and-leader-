# 评教系统 (Course Evaluation System)

## 项目简介
这是一个基于 Vue 3 + Vite (前端) 和 FastAPI (后端) 开发的评教系统。

## 功能特性
- 教师端：查看评价、管理课程
- 管理端：系统配置、用户管理
- 学生端：提交评价

## 快速开始

### 前端启动
```bash
cd 评教系统教师端/frontend_Fn8hD/frontend
npm install
npm run dev
```

### 后端启动
```bash
cd 评教系统教师端/backend
pip install -r requirements.txt
python main.py
```
