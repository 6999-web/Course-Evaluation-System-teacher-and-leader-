## 系统概述

### 系统功能

评教系统自动评分模块是一个基于人工智能的教学评估系统，具有以下核心功能：

- **自动评分**：利用 Deepseek AI 对教师提交的教学文件进行自动评分
- **模板管理**：管理5类教学文件的评分标准模板
- **人工复核**：对自动评分结果进行人工复核和调整
- **异议处理**：处理教师对评分结果的异议申请
- **结果公示**：在所有教师确认后公示最终评分结果
- **数据分析**：统计自动评分与人工复核的一致性

### 支持的文件类型

系统支持以下5类教学文件的评分：

1. **教案** - 教师的课堂教学设计文档
2. **教学反思** - 教师对教学过程的反思总结
3. **教研/听课记录** - 教研活动或听课的记录文档
4. **成绩/学情分析** - 学生成绩或学情的分析报告
5. **课件** - 教学课件（PowerPoint 或其他格式）

### 支持的文件格式

- Word 文档（.docx）
- PDF 文档（.pdf）
- PowerPoint 演示文稿（.pptx）

## 提示词模板配置

### 访问模板配置页面

1. 点击左侧菜单中的"模板配置"
2. 系统显示当前所有模板列表

### 查看模板详情

1. 在模板列表中找到要查看的模板
2. 点击模板名称或"查看"按钮
3. 系统显示模板的详细内容，包括：
   - 等级划分标准
   - 核心指标及其权重
   - 一票否决项规则
   - 加分项规则
   - 输出格式定义

### 编辑模板

#### 修改指标分值

1. 点击要编辑的模板
2. 在"核心指标"部分，修改各指标的分值
3. 确保所有指标的总分为100分
4. 点击"保存"按钮

**注意**：修改指标分值后，新的评分将按照新标准进行，已完成的评分不会自动调整。

#### 启用/禁用否决项

1. 点击要编辑的模板
2. 在"一票否决项"部分，勾选或取消勾选要启用/禁用的否决项
3. 点击"保存"按钮

#### 配置加分项

1. 点击要编辑的模板
2. 在"加分项规则"部分，配置加分项的名称和最大分值
3. 确保所有加分项的总分不超过10分
4. 点击"保存"按钮

### 创建自定义模板

1. 点击"新建模板"按钮
2. 选择文件类型（教案、教学反思等）
3. 填写模板内容：
   - 等级划分标准
   - 核心指标定义
   - 否决项规则
   - 加分项规则
4. 点击"保存"按钮

**提示**：建议基于现有模板进行修改，而不是从零开始创建。

## 考评任务管理

### 创建考评任务

1. 点击左侧菜单中的"考评任务管理"
2. 点击"新建任务"按钮
3. 填写任务信息：
   - **任务名称** - 例如"2024年春季学期教学评估"
   - **任务描述** - 任务的详细说明
   - **截止时间** - 教师提交文件的截止日期和时间
   - **参与教师** - 选择参与此任务的教师

4. 配置文件要求：
   - 勾选需要教师上传的文件类型（可多选）
   - 例如：教案、教学反思、课件

5. 配置评分规则：
   - **启用自动评分** - 勾选此项启用自动评分功能
   - **启用加分项** - 勾选此项允许添加加分项
   - **最大加分值** - 设置加分的上限（默认10分）

6. 点击"创建"按钮完成任务创建

### 查看任务详情

1. 在任务列表中找到要查看的任务
2. 点击任务名称或"查看"按钮
3. 系统显示任务的详细信息和参与教师列表

### 修改任务

1. 在任务列表中找到要修改的任务
2. 点击"编辑"按钮
3. 修改任务信息
4. 点击"保存"按钮

**注意**：任务创建后，如果已有教师提交文件，则不能修改文件类型要求。

### 删除任务

1. 在任务列表中找到要删除的任务
2. 点击"删除"按钮
3. 系统提示确认删除
4. 点击"确认"按钮完成删除

**警告**：删除任务将同时删除所有相关的文件提交和评分记录，此操作不可恢复。

## 自动评分

### 查看待评分文件

1. 在考评任务列表中找到要评分的任务
2. 点击任务行右侧的"操作"菜单
3. 选择"查看文件"
4. 系统显示该任务下所有教师提交的文件列表，包括：
   - 教师姓名
   - 文件类型
   - 提交时间
   - 当前状态（待评分、已评分、评分失败等）

### 触发单个文件评分

1. 在文件列表中找到要评分的文件
2. 点击文件行右侧的"评分"按钮
3. 系统弹出评分对话框，显示：
   - 文件基本信息
   - 文件预览（如果支持）
   - 加分项输入框（如果启用了加分项）

4. 如需添加加分项：
   - 点击"添加加分项"按钮
   - 选择加分项名称
   - 输入加分值（不超过该项的最大值）
   - 输入加分理由或佐证材料

5. 点击"开始评分"按钮
6. 系统调用 AI 进行评分，显示进度条
7. 评分完成后，系统显示评分结果：
   - 基础分
   - 加分项明细
   - 最终得分
   - 等级
   - 扣分理由
   - 改进建议

8. 点击"确认"按钮保存评分结果

### 批量评分

1. 在考评任务列表中找到要评分的任务
2. 点击任务行右侧的"操作"菜单
3. 选择"批量评分"
4. 系统弹出批量评分对话框，显示：
   - 待评分文件数量
   - 预计评分时间
   - 加分项配置

5. 点击"开始评分"按钮
6. 系统开始批量评分，显示进度条
7. 评分过程中可以：
   - 查看当前评分进度
   - 查看已完成的评分结果
   - 暂停评分（点击"暂停"按钮）

8. 所有文件评分完成后，系统显示评分统计：
   - 总文件数
   - 成功评分数
   - 失败评分数
   - 平均得分
   - 等级分布

### 查看评分结果

1. 在考评任务列表中找到已评分的任务
2. 点击任务行右侧的"操作"菜单
3. 选择"查看结果"
4. 系统显示该任务的所有评分结果，包括：
   - 教师姓名
   - 文件类型
   - 得分
   - 等级
   - 评分时间
   - 状态（已评分、待复核、已复核等）

5. 点击某条记录可查看详细的评分明细

### 处理评分失败

如果某个文件评分失败，系统会显示失败原因，可能的原因包括：

- **文件格式不支持** - 文件格式不是 docx、pdf 或 pptx
- **文件内容为空** - 文件没有可读取的内容
- **API 调用失败** - 与 AI 服务的连接出现问题
- **文件解析失败** - 文件损坏或无法读取

**处理方法**：

1. 检查文件是否符合要求
2. 如果文件有问题，通知教师重新上传
3. 如果是 API 问题，稍后重试评分
4. 如果问题持续存在，联系系统管理员

## 评分复核

### 查看异议申请

1. 点击左侧菜单中的"复核管理"
2. 点击"异议申请"标签
3. 系统显示所有异议申请列表，包括：
   - 教师姓名
   - 异议理由
   - 提交时间
   - 当前状态（待复核、复核中、已解决）

### 处理异议申请

1. 在异议申请列表中找到要处理的申请
2. 点击申请行查看详情
3. 系统显示：
   - 原始评分结果
   - 教师的异议理由
   - 文件内容预览

4. 进行人工复核：
   - 仔细阅读文件内容
   - 参考原始评分明细
   - 考虑教师的异议理由

5. 做出复核决定：
   - 如果同意异议，点击"调整评分"按钮
   - 输入新的评分结果
   - 输入复核理由
   - 点击"保存"按钮

   - 如果不同意异议，点击"驳回异议"按钮
   - 输入驳回理由
   - 点击"保存"按钮

6. 系统自动通知教师复核结果

### 随机抽查

1. 点击左侧菜单中的"复核管理"
2. 点击"随机抽查"标签
3. 点击"新建抽查"按钮
4. 填写抽查参数：
   - **任务选择** - 选择要抽查的考评任务
   - **抽查比例** - 设置抽查比例（例如 10%）
   - **抽查数量** - 系统自动计算

5. 点击"开始抽查"按钮
6. 系统随机选取样本并标记为"待复核"状态
7. 对抽取的样本进行人工复核：
   - 点击样本记录
   - 查看原始评分和文件内容
   - 进行人工评分
   - 记录复核结果（一致/不一致）
   - 如果不一致，记录差异原因

### 查看一致性统计

1. 点击左侧菜单中的"复核管理"
2. 点击"一致性统计"标签
3. 系统显示一致性统计报告，包括：
   - 总复核数量
   - 一致数量
   - 不一致数量
   - 一致性比例
   - 差异原因分布

4. 可以按时间范围、任务、文件类型等条件筛选统计结果

## 结果公示

### 公示前的准备

在公示评分结果前，需要完成以下步骤：

1. **完成所有评分** - 所有文件都已评分
2. **处理所有异议** - 所有异议申请都已处理
3. **教师确认** - 所有教师都已确认评分结果

### 查看确认状态

1. 在考评任务列表中找到要公示的任务
2. 点击任务行右侧的"操作"菜单
3. 选择"查看确认状态"
4. 系统显示所有教师的确认状态：
   - 教师姓名
   - 评分状态
   - 确认状态
   - 确认时间

### 公示结果

1. 在考评任务列表中找到要公示的任务
2. 点击任务行右侧的"操作"菜单
3. 选择"公示结果"
4. 系统提示确认公示
5. 点击"确认公示"按钮
6. 系统将评分结果公示，所有教师都可以查看最终结果

**注意**：公示后，评分结果将不能再修改。

### 撤销公示

如果需要撤销已公示的结果（仅在特殊情况下）：

1. 在考评任务列表中找到已公示的任务
2. 点击任务行右侧的"操作"菜单
3. 选择"撤销公示"
4. 系统提示确认撤销
5. 点击"确认撤销"按钮

**警告**：撤销公示是一个严重的操作，应谨慎使用。

## 数据导出

### 导出评分结果

1. 在考评任务列表中找到要导出的任务
2. 点击任务行右侧的"操作"菜单
3. 选择"导出结果"
4. 系统弹出导出对话框，显示导出选项：
   - **导出格式** - 选择 Excel 格式
   - **包含内容** - 选择要导出的字段
   - **筛选条件** - 可选的筛选条件

5. 点击"导出"按钮
6. 系统生成 Excel 文件并下载

### 导出内容

导出的 Excel 文件包含以下列：

- 教师姓名
- 文件类型
- 基础分
- 加分项明细
- 最终得分
- 等级
- 扣分理由
- 改进建议
- 评分时间
- 评分类型（自动/人工）
- 确认状态

### 导出一致性报告

1. 点击左侧菜单中的"复核管理"
2. 点击"一致性统计"标签
3. 点击"导出报告"按钮
4. 系统生成一致性报告 Excel 文件并下载

## 常见问题

### Q1: 如何处理评分失败的文件？

**A**: 评分失败通常是由于以下原因：
- 文件格式不支持（需要是 docx、pdf 或 pptx）
- 文件内容为空或无法读取
- 与 AI 服务的连接问题

**解决方法**：
1. 检查文件格式是否正确
2. 尝试重新上传文件
3. 稍后重试评分
4. 如果问题持续，联系系统管理员

### Q2: 如何修改已评分的结果？

**A**: 有两种方式修改评分结果：

1. **通过异议处理** - 如果教师提交了异议，可以在复核管理中调整评分
2. **直接人工评分** - 在评分结果列表中，点击"重新评分"按钮进行人工评分

### Q3: 加分项有什么限制？

**A**: 加分项的限制如下：
- 每个加分项都有最大分值限制
- 所有加分项的总分不能超过 10 分
- 系统会自动限制超出的加分

### Q4: 如何确保评分的准确性？

**A**: 系统提供了多种机制确保评分准确性：
- 自动评分基于专业的评分标准模板
- 支持人工复核和异议处理
- 支持随机抽查验证一致性
- 生成一致性报告供参考

### Q5: 公示后还能修改评分吗？

**A**: 不能。公示后的评分结果是最终的，不能再修改。如果需要修改，必须先撤销公示，但这是一个严重的操作，应谨慎使用。

### Q6: 如何处理超时未提交的教师？

**A**: 系统会自动对超时未提交的教师触发"一票否决"，判定为不合格。这个过程是自动的，无需管理员手动操作。

### Q7: 一致性比例多少才算合格？

**A**: 系统的目标是达到 95% 的一致性比例。如果一致性比例低于 95%，建议：
1. 检查评分标准模板是否清晰
2. 对评分人员进行培训
3. 调整评分标准或否决项规则

### Q8: 如何导出特定时间范围的评分结果？

**A**: 在导出对话框中，可以设置筛选条件：
1. 选择"按时间范围筛选"
2. 设置开始日期和结束日期
3. 点击"导出"按钮

### Q9: 系统支持哪些浏览器？

**A**: 系统支持以下浏览器：
- Google Chrome（推荐）
- Microsoft Edge
- Mozilla Firefox

### Q10: 如何联系技术支持？

**A**: 如遇到技术问题，请：
1. 查看本手册的常见问题部分
2. 联系学校的 IT 部门
3. 提供错误信息和操作步骤，以便快速解决问题
