## 系统概述

### 系统功能

评教系统是学校用于教学评估的信息化平台。作为教师，您可以通过该系统：

- **查看待办考评** - 查看学校分配给您的考评任务
- **上传教学文件** - 按要求上传教案、教学反思等教学文件
- **查看评分结果** - 查看系统对您提交文件的评分结果
- **提交异议申请** - 对评分结果有异议时提交申请
- **确认评分结果** - 在确认无误后确认最终评分结果

### 评分流程

系统的评分流程如下：

1. **学校发布任务** - 学校管理员创建考评任务并分配给您
2. **您上传文件** - 您按要求上传指定类型的教学文件
3. **系统自动评分** - 系统使用 AI 技术对您的文件进行自动评分
4. **您查看结果** - 您可以查看详细的评分结果和扣分理由
5. **您可提交异议** - 如果对评分有异议，可以提交异议申请
6. **管理员复核** - 管理员对异议进行人工复核
7. **您确认结果** - 在确认无误后确认最终评分结果
8. **学校公示结果** - 所有教师确认后，学校公示最终结果

## 登录和基本操作

### 登录系统

1. 打开浏览器，访问评教系统教师端
2. 输入用户名（通常是工号）和密码
3. 点击"登录"按钮
4. 系统验证成功后进入首页

**提示**：
- 首次登录时，系统会要求您修改初始密码
- 建议使用强密码（包含大小写字母、数字和特殊符号）
- 如忘记密码，点击"忘记密码"链接进行重置

### 首页概览

登录后，您会看到首页，包含以下信息：

- **待办考评** - 显示您需要完成的考评任务数量
- **待确认评分** - 显示您需要确认的评分结果数量
- **待处理异议** - 显示您提交的异议申请状态
- **快速操作** - 提供常用功能的快捷入口

### 导航菜单

教师端主菜单包含以下选项：

- **首页** - 系统概览和快速操作
- **待办考评** - 查看和处理待办考评任务
- **我的评分** - 查看所有评分结果
- **异议申请** - 查看和管理异议申请
- **个人设置** - 修改个人信息和密码

### 修改个人信息

1. 点击右上角的用户头像或用户名
2. 选择"个人设置"
3. 可以修改以下信息：
   - 邮箱地址
   - 联系电话
   - 个人简介

4. 点击"保存"按钮

### 修改密码

1. 点击右上角的用户头像或用户名
2. 选择"修改密码"
3. 输入当前密码
4. 输入新密码（至少8个字符，包含大小写字母、数字）
5. 确认新密码
6. 点击"保存"按钮

## 查看待办考评

### 访问待办考评页面

1. 点击左侧菜单中的"待办考评"
2. 系统显示所有分配给您的考评任务列表

### 理解任务信息

任务列表中的每条记录包含以下信息：

- **任务名称** - 考评任务的名称
- **任务描述** - 任务的详细说明
- **截止时间** - 提交文件的截止日期和时间
- **需要上传的文件** - 列出需要上传的文件类型
- **状态** - 任务的当前状态（未开始、进行中、已完成、已逾期）

### 查看任务详情

1. 在任务列表中找到要查看的任务
2. 点击任务名称或"查看详情"按钮
3. 系统显示任务的详细信息，包括：
   - 完整的任务描述
   - 需要上传的文件类型及说明
   - 截止时间倒计时
   - 您已上传的文件列表
   - 已评分文件的评分结果

### 了解文件要求

在任务详情页面，您可以看到需要上传的文件类型。每种文件类型都有以下说明：

- **文件类型** - 例如"教案"、"教学反思"等
- **文件格式** - 支持的格式（docx、pdf、pptx）
- **文件说明** - 对文件内容的要求和建议
- **评分标准** - 该文件类型的评分标准概览

## 上传教学文件

### 上传文件的步骤

1. 在待办考评页面找到要上传文件的任务
2. 点击"上传文件"按钮
3. 系统弹出文件上传对话框
4. 选择要上传的文件类型
5. 点击"选择文件"按钮，从计算机中选择文件
6. 确认文件信息无误
7. 点击"上传"按钮

### 文件要求

上传文件时，请注意以下要求：

- **文件格式** - 必须是 docx、pdf 或 pptx 格式
- **文件大小** - 单个文件不超过 50MB
- **文件内容** - 文件必须有实质性内容，不能为空
- **文件名** - 建议使用有意义的文件名，例如"2024春季教案_数学"

### 上传成功

文件上传成功后，系统会显示：

- 上传成功的提示信息
- 文件的基本信息（文件名、大小、上传时间）
- 文件的当前状态（通常为"待评分"）

系统会自动对上传的文件进行评分。评分通常在几分钟内完成。

### 上传失败

如果文件上传失败，系统会显示错误信息。常见的错误包括：

- **文件格式不支持** - 请确保文件是 docx、pdf 或 pptx 格式
- **文件大小超限** - 请压缩文件或分割成多个文件
- **文件内容为空** - 请确保文件有实质性内容
- **网络连接问题** - 请检查网络连接并重试

### 重新上传文件

在截止时间前，您可以重新上传文件来替换之前上传的版本：

1. 在任务详情页面找到已上传的文件
2. 点击文件行右侧的"重新上传"按钮
3. 选择新的文件
4. 点击"上传"按钮
5. 系统会用新文件替换旧文件，并重新进行评分

**注意**：重新上传后，之前的评分结果会被覆盖。

### 删除文件

如果需要删除已上传的文件（在截止时间前）：

1. 在任务详情页面找到要删除的文件
2. 点击文件行右侧的"删除"按钮
3. 系统提示确认删除
4. 点击"确认"按钮完成删除

**注意**：删除文件后，相关的评分记录也会被删除。

## 查看评分结果

### 访问评分结果页面

1. 点击左侧菜单中的"我的评分"
2. 系统显示所有您提交的文件的评分结果

或者：

1. 在待办考评页面找到已评分的任务
2. 点击"查看结果"按钮

### 理解评分结果

每条评分记录包含以下信息：

- **文件信息** - 文件类型、提交时间
- **基础分** - 根据评分标准计算的基础分数
- **加分项** - 如果有加分项，会显示加分项名称和分值
- **最终得分** - 基础分加上加分项的总分
- **等级** - 根据得分确定的等级（优秀、良好、合格、不合格）
- **扣分理由** - 详细说明为什么扣分
- **改进建议** - 提出的改进建议

### 查看详细评分明细

1. 在评分结果列表中找到要查看的记录
2. 点击记录行或"查看详情"按钮
3. 系统显示详细的评分明细，包括：
   - 各个评分指标的得分
   - 每个指标的评分理由
   - 是否触发了否决项
   - 完整的评分报告

### 理解等级标准

系统使用以下等级标准：

- **优秀** - 得分 90-100 分
- **良好** - 得分 80-89 分
- **合格** - 得分 60-79 分
- **不合格** - 得分低于 60 分

### 下载评分报告

1. 在评分结果详情页面
2. 点击"下载报告"按钮
3. 系统生成 PDF 格式的评分报告并下载

## 提交异议申请

### 何时提交异议

如果您对评分结果有异议，可以提交异议申请。常见的异议原因包括：

- 认为评分标准理解有误
- 认为评分人员没有充分理解文件内容
- 认为扣分理由不合理
- 认为评分结果与实际情况不符

### 提交异议的步骤

1. 在评分结果详情页面
2. 点击"提交异议"按钮
3. 系统弹出异议申请对话框
4. 在"异议理由"文本框中详细说明您的异议原因
5. 可以选择上传补充材料（可选）
6. 点击"提交"按钮

### 异议理由的建议

提交异议时，请提供清晰、具体的理由：

- **具体指出问题** - 例如"第二个指标的评分理由不准确"
- **提供证据** - 引用文件中的具体内容来支持您的观点
- **保持专业态度** - 使用礼貌、专业的语言
- **避免过长** - 简明扼要地说明问题，通常 200-500 字为宜

### 异议申请的状态

提交异议后，您可以查看异议的处理状态：

- **待复核** - 异议已提交，等待管理员复核
- **复核中** - 管理员正在复核您的异议
- **已解决** - 管理员已完成复核，有了复核结果

### 查看复核结果

1. 点击左侧菜单中的"异议申请"
2. 找到已解决的异议申请
3. 点击"查看结果"按钮
4. 系统显示管理员的复核结果，包括：
   - 是否同意异议
   - 复核理由
   - 调整后的评分结果（如果有调整）

### 异议处理时间

系统规定管理员应在 3 个工作日内完成异议复核。如果超过此时间，您可以联系学校管理部门跟进。

## 确认评分结果

### 何时需要确认

在以下情况下，您需要确认评分结果：

- 初次查看评分结果时
- 异议被复核后，有了新的评分结果时
- 管理员进行了人工调整时

### 确认评分的步骤

1. 在评分结果详情页面
2. 仔细查看评分明细和扣分理由
3. 如果确认无误，点击"确认评分"按钮
4. 系统提示确认
5. 点击"确认"按钮完成确认

### 确认前的检查清单

在确认评分前，请检查以下内容：

- [ ] 文件类型是否正确
- [ ] 得分是否合理
- [ ] 扣分理由是否清晰
- [ ] 是否有任何疑问需要澄清
- [ ] 是否需要提交异议

### 确认后的影响

确认评分后：

- 您的评分结果将被锁定，不能再提交异议
- 您的确认状态会显示在管理员的公示前检查中
- 当所有教师都确认后，管理员可以公示最终结果

### 修改已确认的评分

如果您已经确认了评分，但后来发现有问题，可以：

1. 联系学校管理部门
2. 说明需要修改的原因
3. 管理员可以撤销您的确认，允许您重新提交异议

## 常见问题

### Q1: 我忘记了登录密码怎么办？

**A**: 您可以通过以下方式重置密码：

1. 在登录页面点击"忘记密码"链接
2. 输入您的用户名或邮箱地址
3. 系统会发送重置链接到您的邮箱
4. 点击邮件中的链接，设置新密码
5. 使用新密码登录

如果您没有收到重置邮件，请检查垃圾邮件文件夹，或联系学校 IT 部门。

### Q2: 上传的文件为什么显示"评分失败"？

**A**: 文件评分失败通常是由于以下原因：

- **文件格式不支持** - 请确保文件是 docx、pdf 或 pptx 格式
- **文件内容为空** - 请确保文件有实质性内容
- **文件损坏** - 请尝试重新保存文件后重新上传
- **系统问题** - 请稍后重试，或联系 IT 部门

**解决方法**：
1. 检查文件格式和内容
2. 尝试重新上传文件
3. 如果问题持续，联系学校 IT 部门

### Q3: 评分结果什么时候会出现？

**A**: 系统通常在文件上传后的几分钟内完成评分。具体时间取决于：

- 文件大小
- 系统负载
- 网络连接速度

通常情况下，评分会在 5-10 分钟内完成。如果超过 30 分钟还没有评分结果，请刷新页面或联系 IT 部门。

### Q4: 我对评分结果有异议，应该怎么做？

**A**: 如果您对评分结果有异议，可以：

1. 在评分结果详情页面点击"提交异议"按钮
2. 详细说明您的异议原因
3. 提交异议申请
4. 等待管理员的复核结果（通常 3 个工作日内）
5. 查看复核结果并确认

### Q5: 我可以重新上传文件吗？

**A**: 可以。在截止时间前，您可以重新上传文件来替换之前的版本：

1. 在任务详情页面找到已上传的文件
2. 点击"重新上传"按钮
3. 选择新的文件
4. 点击"上传"按钮

重新上传后，系统会用新文件替换旧文件，并重新进行评分。

### Q6: 截止时间过了还能上传文件吗？

**A**: 不能。截止时间过后，系统会自动关闭上传功能。如果您有特殊原因需要延期上传，请：

1. 立即联系学校管理部门
2. 说明延期的原因
3. 请求管理员延长截止时间

### Q7: 我确认了评分后还能改吗？

**A**: 确认后，您不能直接修改评分。但如果您发现有问题，可以：

1. 联系学校管理部门
2. 说明需要修改的原因
3. 请求管理员撤销您的确认
4. 重新提交异议申请

### Q8: 评分结果会公示吗？

**A**: 会的。当所有教师都确认评分结果后，学校管理员会公示最终的评分结果。公示后，所有教师都可以查看整体的评分情况。

### Q9: 我可以查看其他教师的评分结果吗？

**A**: 不能。系统只允许您查看自己的评分结果。其他教师的评分结果是保密的，只有学校管理部门可以查看。

### Q10: 系统支持哪些浏览器？

**A**: 系统支持以下浏览器：

- Google Chrome（推荐）
- Microsoft Edge
- Mozilla Firefox

建议使用最新版本的浏览器以获得最佳体验。

### Q11: 我的文件被标记为"未提交核心文件"是什么意思？

**A**: 这意味着系统检测到您的文件内容为空或无法读取。可能的原因包括：

- 文件确实是空的
- 文件格式不正确
- 文件损坏

**解决方法**：
1. 检查文件内容是否为空
2. 确保文件格式正确（docx、pdf 或 pptx）
3. 尝试重新保存文件
4. 重新上传文件

### Q12: 如何联系技术支持？

**A**: 如遇到技术问题，请：

1. 查看本手册的常见问题部分
2. 联系学校的 IT 部门
3. 提供以下信息以便快速解决问题：
   - 您的用户名
   - 问题描述
   - 错误信息（如果有）
   - 操作步骤

## 提示和建议

### 提高评分的建议

1. **认真准备文件** - 确保文件内容完整、准确、专业
2. **遵循评分标准** - 了解评分标准，按照标准准备文件
3. **及时上传** - 不要等到截止时间最后一刻才上传
4. **保留备份** - 上传前保留文件的备份副本
5. **仔细检查** - 上传前检查文件格式和内容

### 提交异议的建议

1. **冷静思考** - 不要在情绪激动时提交异议
2. **充分准备** - 准备充分的证据和理由
3. **专业态度** - 使用礼貌、专业的语言
4. **及时提交** - 在规定时间内提交异议
5. **保持沟通** - 与管理部门保持良好沟通

### 系统使用的建议

1. **定期检查** - 定期登录系统检查新的任务和消息
2. **及时确认** - 及时确认评分结果，不要拖延
3. **保持更新** - 及时更新个人信息和联系方式
4. **学习帮助** - 充分利用系统的帮助文档和常见问题
5. **反馈意见** - 如有建议，可以向管理部门反馈
