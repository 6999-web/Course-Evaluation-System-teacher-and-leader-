- **评定等级**：优秀/良好/合格/不合格
- **详细评分**：各维度的具体得分和理由
- **AI总结**：专业的评价和改进建议

## 🔧 故障排除

### 常见问题及解决方案

#### 1. 后端启动失败
**问题**：`ImportError: attempted relative import with no known parent package`
**解决**：
```bash
# 确保在正确目录运行
cd 评教系统最终版/评教系统管理端/backend_8fMBP/backend
python -c "import uvicorn; uvicorn.run('app.main:app', host='localhost', port=8001)"
```

#### 2. 评分失败
**问题**：`文件不存在`错误
**解决**：
- 检查文件路径是否正确
- 确保文件已正确上传到服务器
- 验证文件格式是否支持

#### 3. API调用超时
**问题**：DeepSeek API调用超时
**解决**：
- 检查网络连接
- 验证API密钥是否有效
- 适当增加超时时间

#### 4. 前端无法访问
**问题**：页面无法加载
**解决**：
```bash
# 检查端口是否被占用
netstat -an | findstr :3000
netstat -an | findstr :8001

# 重启服务
npm run dev
```

## 📈 性能优化建议

### 1. 批量评分优化
- 建议每批处理不超过10个文件
