# 评教系统快速启动指南

## 系统要求

- Python 3.10+
- Node.js 16+
- npm 或 yarn
- Windows/Linux/macOS

## 快速启动（5分钟）

### 1. 启动教师端前端
```bash
cd 评教系统最终版/评教系统教师端/frontend
npm run dev
```
访问: http://localhost:5173

### 2. 启动教师端后端
```bash
cd 评教系统最终版/评教系统教师端/backend
python main.py
```
API: http://localhost:8000/api

### 3. 启动管理端前端
```bash
cd 评教系统最终版/评教系统管理端/frontend_Fn8hD/frontend
npm run dev
```
访问: http://localhost:5174

### 4. 启动管理端后端
```bash
cd 评教系统最终版/评教系统管理端/backend_8fMBP/backend
uvicorn app.main:app --reload --port 8001
```
API: http://localhost:8001/api

## 常见问题

### Q: 端口已被占用
A: 修改启动命令中的端口号，例如：
```bash
npm run dev -- --port 5175
uvicorn app.main:app --reload --port 8002
```

### Q: 模块导入错误
A: 确保已安装所有依赖：
```bash
pip install -r requirements.txt
npm install
```

### Q: 数据库连接失败
A: 检查SQLite数据库文件是否存在，如不存在会自动创建。

### Q: 前端无法连接后端
A: 检查CORS配置，确保后端允许前端域名访问。

## 故障排查

### 教师端后端无法启动
1. 检查Python版本: `python --version`
2. 检查依赖: `pip list | grep fastapi`
3. 查看错误日志

### 管理端后端无法启动
1. 检查导入路径
2. 检查数据库连接
3. 查看错误日志

### 前端无法加载
1. 检查Node.js版本: `node --version`
2. 清除缓存: `npm cache clean --force`
3. 重新安装依赖: `npm install`
